How to work with lists

Working with ordered lists

First step: In order to create a list of items to be followed in an order. one must use the 'ol' tag.

How to order the items: In order to order said items, one must tag the tag 'li', add the item and then close the tag. Those tags will appear in the order they were added top to bottom in the code.

Working with unordered lists

First step: The process for using unordered lists is pretty similar to that of using ordered lists, with the exception that one must use the 'ul' tag.

How to order the items: The process here is precisely the same as in ordered lists using the li tag.
